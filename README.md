# Destruction-Derby

## Controls

### Keyboard:

Player 1:

1: Respawn
W: Speed up
S: Speed down
A: Left
D: Right
Lshift: Hand brake
Lctrl: Shoot

Player 2:

0: Respawn
Up arrow: Speed up
Down arrow: Speed down
Left arrow: Left
Right arrow: Right
Rshift: Hand brake
Rctrl: Shoot

Debugs buttons:

F1: Debug draw
F4: Debug camera

Camera debug:

I: Forward
K: Backward
J: Left
L: Right
O: Up
P: Down

[ mouse right click to rotate ]
